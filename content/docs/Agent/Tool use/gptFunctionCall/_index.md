# Function Call
### 调用顺序  [0] [1][2]

+ Function Calling 整个功能的调用顺序大致如下
  - 声明函数：定义当前函数的名称，描述，以及对应的参数信息，并请求对应的接口；
  - 解析函数参数：接受对应的接口返回，并解析对应的函数参数信息；
  - 执行函数：根据对应的参数信息调用本地函数；
  - 上报结果：将本地函数执行的结果上报给 Chat 接口；
